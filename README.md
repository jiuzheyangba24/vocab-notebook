# vocab-notebook · 英语四级/六级 & 考研词汇生词本

面向英语四级（CET-4/6）与考研英语的词汇学习与测验应用。支持后端持久化、错题本复习、考试风格的测试模式、统计与键盘快捷操作，帮助你高效建立与巩固考试所需词汇量。

## 功能概览
- 生词管理：查看当前词、上一词/下一词、单个添加、**批量添加**、按 `id` 删除。
- **批量添加**：支持一次性添加多个单词，自动获取释义、例句和同义词，带进度显示和错误处理。
- 后端 API：Node.js + Express 提供词库读写（JSON 文件持久化）。
- 测试模式：英译中 / 中译英 / 填空测试 / 单词拼写。
- 测试设置：可选择测试模式、题目数量，并支持"仅测错题"。
- 错题本：自动记录错题，支持查看与一键清空。
- 统计与反馈：每题即时反馈；结束时显示答对数、答错数、正确率与总题数。
- 键盘快捷键：
  - 空格键：在测试中智能触发"显示答案/下一题"，拼写模式会自动聚焦输入框。
  - 回车键：拼写模式提交答案。

## 适用场景（CET-4/6 与考研英语）
- 四六级备考：通过“英译中/中译英/填空/拼写”覆盖词义识别与拼写能力，适配客观题风格。
- 考研英语：利用“错题本 + 仅测错题”强化薄弱词，结合拼写模式提升主动回忆能力。
- 日常巩固：设置每次 10–30 题的短测，完成后查看正确率并针对性复习。

## 推荐使用方式（备考建议）
- 每日小测：`10–20` 题，优先勾选“仅测错题”，持续拉升正确率。
- 模式轮换：在“英译中/中译英/拼写”之间交替，兼顾识别与输出。
- 错题再战：错题进入错题本后，定期进行“仅测错题”专项训练，直到正确率 > 90%。
- 间隔复习：可按 1 天、3 天、7 天的间隔重复测错题（艾宾浩斯思路）。

## 使用说明

### 添加单词
- **单个添加**：在输入框输入单词，点击"添加生词"按钮，系统自动获取释义、例句和同义词。
- **批量添加**：
  1. 点击"批量添加"按钮打开批量添加窗口
  2. 在文本框中输入单词，每行一个（例如：hello、world、vocabulary）
  3. 点击"开始添加"按钮
  4. 系统会自动逐个获取单词信息并添加到词库
  5. 实时显示添加进度和结果（成功/跳过/失败）
  6. 已存在的单词会自动跳过，避免重复

### 测试功能
- 在"测试设置"中选择测试模式与题目数量。
- 可勾选"仅测错题"从错题本抽题（自动去重并按唯一词数限制题量）。
- 测试时：
  - 选择题直接点击选项；
  - 拼写模式在输入框输入英文单词并"检查"，支持回车提交；
  - 按空格键可快速显示答案并进入下一题。
- 测试结束会显示详细统计；错题会记录到本地浏览器 `localStorage` 的 `wrongQuestions`。

## 后端 API
- `GET /api/words`：获取全部单词。
- `GET /api/words/count`：获取词数 `{ count: number }`。
- `POST /api/words`：新增单词。
- `DELETE /api/words/:id`：按 `id` 删除单词。

### 数据结构示例（`vocabulary.json`）
```json
[
  {
    "id": "uuid-or-number",
    "headWord": "hello",
    "definition": "喂；你好",
    "sentences": ["Hello, John! How are you?"],
    "synonyms": ["hallo"],
    "pronunciation": "hə'ləʊ"
  }
]
```

## 常见问题
- 无法开始选择题测试：请确保题库（或错题池）至少包含 4 个单词。
- 删除失败：后端删除接口使用 `id`，前端需要传递 `word.id`，不是 `headWord`。
- 计数接口路径：前端应请求 `GET /api/words/count`（不是 `/api/word-count`）。
- 端口占用：默认端口为 `3000`，如需修改可在 `server.js` 中调整。
- 批量添加速度慢：为避免API限流，每个单词之间有500ms延迟，这是正常现象。
- 批量添加部分失败：可能是网络问题或API限制，可以重试失败的单词。
